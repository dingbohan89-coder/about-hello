# 选题说明

选题池用于存放还没有验证的内容想法。不要把完整文案写在这里，只记录选题、目标、平台和测试结果。

## 内容类型

```text
功能体验
品牌解释
活动提醒
数据揭秘
情绪共鸣
用户裂变
用户故事
安全信任
```

## 优先级

```text
A：本周优先测试
B：近期可以测试
C：备用选题
D：暂不测试
```

## 状态

```text
待测试
本周测试
已发布
有效
一般
无效
归档
```

## 选题池和其他目录的关系

```text
选题池 = 想法仓库
周计划 = 本周作战表
内容项目 = 已经制作的内容档案
爆款复盘 = 被验证有效的方法
```

## 使用流程

1. 每周从 `选题池.csv` 里挑 3-5 个 A 级选题。
2. 写入 `04_周计划/本周策略.md`。
3. 一次性生成各平台文案。
4. 按 `发布排期.csv` 发布。
5. 周末把测试结果回填到 `选题池.csv`。
6. 表现特别好的内容，沉淀到 `00_总览/爆款复盘.md`。
