# 哈喽Hello 素材库说明

这是哈喽Hello的统一内容素材库，用于管理小红书、公众号、抖音、朋友圈、微信社群等平台的文案、提示词、成品素材和复盘记录。

## 使用原则

1. 所有具体发布内容都放到 `02_内容项目` 对应平台文件夹。
2. 每个平台下按 `YYYY-MM-DD_主题_风格` 创建项目文件夹。
3. 一个内容项目固定包含：`brief.md`、`文案.md`、`提示词.md`、`成品`、`复盘.md`。
4. 发布后登记到 `00_总览/内容总表.csv`。
5. 有明确发布时间的内容登记到 `00_总览/发布日历.csv`。
6. 表现特别好或特别差的内容，再总结到 `00_总览/爆款复盘.md`。
7. AI 生成内容前，先读取 `01_品牌资产` 和对应项目的 `brief.md`。

## 项目命名

推荐格式：

```text
YYYY-MM-DD_主题_风格
```

示例：

```text
2026-06-14_测测我俩_轻松吐槽
2026-06-15_周一匹配预告_朋友聊天
2026-06-16_协作者招募_校园真实感
```

## 项目五件套

```text
brief.md     这条内容为什么做、给谁看、希望用户做什么
文案.md      最终文案、脚本、标题、字幕
提示词.md    AI 生成图片或视频时使用的提示词
成品         图片、视频、截图等最终素材
复盘.md      发布后的数据、反馈、结论和下一版优化
```
