# 素材库说明

这个库只保留六个常用模块，避免内容越来越杂。

## 模块说明

| 模块 | 用途 |
|---|---|
| `00_总览` | 内容总表、爆款复盘、素材库说明 |
| `01_品牌资产` | 品牌定位、支持学校、视觉规范 |
| `02_内容项目` | 按平台保存已经制作或发布的内容 |
| `03_模板` | 少量高频模板，避免重复造格式 |
| `04_周计划` | 一次性规划一周发布策略和文案 |
| `05_选题池` | 保存待测试选题和测试结果 |
| `99_归档` | 旧资料、旧模板、不常用内容 |

## 推荐工作流

1. 先在 `05_选题池/选题池.csv` 里记录想法。
2. 每周挑选 3-5 个选题，进入 `04_周计划`。
3. 一次性写好本周各主平台文案：公众号、小红书、抖音、朋友圈。
4. 发布后把数据记入 `00_总览/内容总表.csv`。
5. 表现好的内容再沉淀到 `00_总览/爆款复盘.md`。

## 周计划发布规则

- 周一：公众号、小红书、抖音、朋友圈各发 2 条图文：匹配预告 + 匹配结果揭秘。
- 周四：公众号、小红书、抖音、朋友圈各发 2 条图文：匹配预告 + 匹配结果揭秘。
- 周二、周三、周五、周六、周日：公众号、小红书、抖音、朋友圈各发 1 条图文。
- 微信群、用户分发、协作者转发等是补充分发素材，可以基于同一周选题多生成不同话术。

## 主平台文案格式

主平台文案文件中，每条内容必须包含：

```text
标题
正文
图片/海报提示词
```

不要单独维护 `海报提示词.md`。图片或海报提示词要直接写在对应平台文案的对应内容下面。

## 什么时候创建内容项目

只有当一条内容已经进入制作或发布阶段时，才在 `02_内容项目` 里建文件夹。

不要把还没确定要做的想法放进 `02_内容项目`，那些应该放在 `05_选题池`。
