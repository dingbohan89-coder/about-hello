# 哈喽Hello 内容素材库

这是哈喽Hello的内容协作仓库，用于管理选题、周计划、平台文案、AI 提示词、成品素材和发布复盘。

## 最常用入口

- `素材库/05_选题池/选题池.csv`：存放待测试选题。
- `素材库/04_周计划`：每周一次性安排发布策略和多平台文案。
- `素材库/02_内容项目`：沉淀已经制作或发布的单条内容。
- `素材库/00_总览/内容总表.csv`：所有内容的总索引和数据记录。
- `素材库/00_总览/爆款复盘.md`：只记录值得复用的结论。

## 日常流程

1. 从 `05_选题池` 选 3-5 个本周要测试的选题。
2. 在 `04_周计划` 新建一周作战表，一次性写好多平台文案。
3. 每天按周计划复制发布到抖音、小红书、朋友圈、微信群等平台。
4. 发布后把数据记到 `00_总览/内容总表.csv`。
5. 有明显效果的内容，沉淀到 `00_总览/爆款复盘.md`。

## 推荐结构

```text
素材库
├─ 00_总览          总索引、爆款复盘、使用说明
├─ 01_品牌资产      品牌定位、学校列表、视觉规范
├─ 02_内容项目      按平台保存已制作内容
├─ 03_模板          少量高频模板
├─ 04_周计划        每周发布作战表
├─ 05_选题池        待测试选题
└─ 99_归档          旧资料和不常用内容
```

## 命名规范

内容项目：`YYYY-MM-DD_主题_风格`

周计划：`YYYY-W周数_内容作战表`

示例：

```text
2026-06-14_测测我俩_轻松吐槽
2026-W25_内容作战表
```
