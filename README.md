# 哈喽Hello 内容素材库

这是哈喽Hello的内容协作仓库，用于管理选题、周计划、平台文案、AI 提示词、成品素材和发布复盘。

## 最常用入口

- `素材库/05_选题池/选题池.csv`：存放待测试选题。
- `素材库/04_周计划`：每周一次性安排发布策略和多平台文案。
- `素材库/02_内容项目`：沉淀已经制作或发布的单条内容。
- `素材库/00_总览/内容总表.csv`：所有内容的总索引和数据记录。
- `素材库/00_总览/爆款复盘.md`：只记录值得复用的结论。

## 日常流程

1. 从 `05_选题池` 选 3-5 个本周要测试的选题。
2. 在 `04_周计划` 新建一周作战表，一次性写好多平台文案。
3. 每天按周计划复制发布到公众号、小红书、抖音、朋友圈等主平台。
4. 发布后把数据记到 `00_总览/内容总表.csv`。
5. 有明显效果的内容，沉淀到 `00_总览/爆款复盘.md`。

## 周计划发布规则

主平台包括：公众号、小红书、抖音、朋友圈。

每周内容默认按这个节奏生成：

- 周一：每个主平台发布 2 条图文，分别是“匹配预告”和“匹配结果揭秘”。
- 周四：每个主平台发布 2 条图文，分别是“匹配预告”和“匹配结果揭秘”。
- 周二、周三、周五、周六、周日：每个主平台每天发布 1 条图文。

微信群、用户分发、协作者转发等不是主平台排期，它们是基于同一周选题生成的补充分发素材，可以多生成几种话术，按场景使用。

## 平台文案文件格式

主平台文案分别写在：

```text
公众号文案.md
小红书文案.md
抖音文案.md
朋友圈文案.md
```

每条内容必须直接包含：

```text
标题
正文
图片/海报提示词
```

不要再单独创建 `海报提示词.md`。海报或配图提示词必须跟对应平台、对应内容放在一起，方便人和 AI 一次性读取、发布和迭代。

推荐结构：

```md
## 2026-06-15 周一匹配预告

### 标题

### 正文

### 图片/海报提示词
```

## 推荐结构

```text
素材库
├─ 00_总览          总索引、爆款复盘、使用说明
├─ 01_品牌资产      品牌定位、学校列表、视觉规范
├─ 02_内容项目      按平台保存已制作内容
├─ 03_模板          少量高频模板
├─ 04_周计划        每周发布作战表
├─ 05_选题池        待测试选题
└─ 99_归档          旧资料和不常用内容
```

## 命名规范

内容项目：`YYYY-MM-DD_主题_风格`

周计划：`YYYY-W周数_内容作战表`

示例：

```text
2026-06-14_测测我俩_轻松吐槽
2026-W25_内容作战表
```
